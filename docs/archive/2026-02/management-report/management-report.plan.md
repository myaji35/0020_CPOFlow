## 개요
- **Feature**: management-report
- **작성일**: 2026-02-28
- **우선순위**: High
- **예상 규모**: Medium (컨트롤러 + 뷰 중심 개선)

---

## 현황 분석

### 현재 구현 (`app/controllers/reports_controller.rb`)
- 월별 수주 트렌드 (최근 12개월) — bar chart (HTML/CSS)
- 발주처별 수주액 Top 10 — horizontal bar
- 거래처별 발주 건수 Top 10 — horizontal bar
- 상태별 파이프라인 현황 — horizontal bar
- 이번달 KPI 카드 6개 (수주·납품·수주액·납기준수율·지연·긴급)
- 납기 위험도 분포 (critical/high/medium/low)

### 현재 문제점
1. **기간 필터 없음** — 이번달 고정, 분기/연간/사용자 지정 불가
2. **차트 빈약** — HTML/CSS 바차트, 선 그래프/파이차트 없음
3. **CSV 내보내기 없음** — 데이터 추출 불가
4. **담당자별 성과 없음** — 직원별 처리 건수/납기 준수율 누락
5. **프로젝트별 분석 없음** — 현장별 수주·납품 현황 누락
6. **전년도 비교 없음** — YoY 성장률 표시 없음
7. **인쇄/PDF 없음** — 경영진 보고용 출력 불가

---

## 기능 요구사항 (FR)

### FR-01: 기간 필터
- 이번달 / 지난달 / 이번 분기 / 올해 / 직접 입력 (from~to)
- URL 파라미터로 상태 유지 (`?period=this_month`)
- 필터 변경 시 모든 데이터 동시 업데이트 (Turbo Frame)

### FR-02: 핵심 KPI 카드 강화
- 기존 6개 + 전월 대비 증감률 (▲▼ 화살표 + %)
- 수주액 합계 (estimated_value 기준)
- 평균 처리 소요일 (inbox→delivered 평균 일수)

### FR-03: 월별 트렌드 차트 (Chart.js)
- 수주 건수 + 납품 건수 이중 선 그래프
- 수주액 막대 그래프 (보조 y축)
- Chart.js CDN 사용 (빌드 없음)

### FR-04: 파이프라인 퍼널 차트
- 7단계 상태별 건수 + 비율 시각화
- 가로형 퍼널 (단계별 색상 구분)

### FR-05: 발주처 / 거래처 / 프로젝트 Top 10
- 기존 수평 바차트 유지 + 기간 필터 연동
- 발주처: 수주액 / 거래처: 발주 건수 / 프로젝트: 수주액

### FR-06: 담당자별 성과
- 직원별 처리 건수 / 납기 준수율 / 평균 소요일
- Order의 user_id 기준 집계

### FR-07: CSV 내보내기
- 현재 필터 기준 전체 데이터 CSV 다운로드
- 컬럼: 주문번호, 발주처, 거래처, 품목, 수주액, 상태, 납기일, 담당자

### FR-08: 인쇄 최적화
- `window.print()` 버튼
- Print CSS: 사이드바 숨김, 페이지 분리, A4 맞춤

## 비기능 요구사항

- **응답 속도**: 필터 변경 후 1초 이내 렌더링
- **권한**: admin + manager만 접근 (기존 유지)
- **다크모드**: 기존 dark: 클래스 패턴 유지
- **Chart.js**: CDN 로드, 빌드 없음
- **Turbo Frame**: 필터 변경 시 페이지 전체 새로고침 없이 데이터 업데이트
